地面戰技巧 (地堡任務)

1. 接受保全任務 (Mercenary)

按 F1 打開 MobiGlas，進入 Contract Manager (合約管理器)。在 Mercenary (傭兵) 標籤下，尋找 "Defend Site" 或 "Provide Backup" 等字眼的任務。這些通常被玩家稱為「地堡任務 (Bunker)」。這類任務風險相對較低，且能穩定賺取 aUEC 並搜刮敵方裝備。

2. 敵我識別 (IFF)

進入地堡後，千萬不要見人就開火！

  • 友軍 (當地保全): 頭頂上通常會有藍色倒三角形標記，穿著該星球專屬顏色的護甲 (例如微科星是白色/灰色，十字軍星是藍色)。
  • 敵軍 (九尾海盜 Nine Tails 等): 沒有標記，護甲顏色多為紫紅色、黑色等混搭風格。
  • 如果不小心擊殺了友軍，您將獲得 CrimeStat (犯罪等級)，任務會直接失敗，並且會被防空炮塔攻擊。

3. 基本戰鬥走位與掩護

星際公民的 AI 雖然有時會卡頓，但準度非常高。切勿站在空曠處與敵人對槍。

  • Q / E: 左右探頭 (Leaning)。利用牆角作為掩體，只露出頭部進行射擊，能大幅降低受傷機率。
  • Ctrl / C: 蹲下與趴下。降低身姿能減少受彈面積並提高射擊穩定度。
  • Shift: 衝刺。在空曠處移動時保持衝刺，尋找下一個掩體。

4. 醫療與倒地處理

受傷時如果出現流血狀態 (血滴圖示)，生命值會不斷流失，視野會變模糊。

立刻按 C 鍵給自己打一針紅色的 Medpen 止血並恢復生命。如果您的肢體受到重創 (顯示紅色或黑色)，醫療針只能暫時緩解，您需要使用醫療槍 (Medical Gun) 或是返回醫院躺醫療床才能徹底治癒。如果您倒地 (Incapacitated)，可以按 M 發出醫療救援信號 (Beacon)，等待其他玩家來救您，或者長按 Backspace 選擇自殺重生 (會遺失身上所有裝備)。
