如何安裝繁體中文化

星際公民目前尚未內建官方中文支援。這是我自己翻譯的雙語中文化專案，目前仍在持續改善中，希望能幫助各位玩家更順利地體驗遊戲。您可以透過以下連結取得：

🔗 專案連結：Star Citizen 繁體雙語中文化 (GitHub)

  1. 前往上方的 GitHub 連結下載最新版的繁體中文化檔案 (global.ini)。
  2. 將該檔案放置於您的遊戲安裝目錄下：
    StarCitizen\LIVE\data\Localization\chinese_(traditional) (如果資料夾不存在請自行建立)
  3. 在StarCitizen\LIVE\下需加入一個 user.cfg，裡面需要有一行g_language = chinese_(traditional)
  4. 重新啟動遊戲，介面即可顯示繁體中文。

中文字元轉換器 Pro

由於遊戲內的聊天視窗無法直接輸入中文字元，我們為您準備了專屬的轉換工具。只要在下方輸入中文，它會自動將其轉碼為 UEE 標準字元，您只需複製後在遊戲中按 Enter 貼上即可！
