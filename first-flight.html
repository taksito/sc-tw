第一次航行操作手冊
(主城至空間站)

1. 在居住艙醒來

按住 Y 鍵可以從床上起身。您可以按 F4 切換第三人稱視角，按住 Z 自由觀看周遭。長按 F 鍵可以呼叫出互動選單，這是遊戲中最重要的一個按鍵，用來開門、按電梯、撿取物品等。

2. 搭乘大眾運輸前往宇航中心

離開居住區後，尋找標有 Transit (交通) 或 Spaceport (太空港) 的指示牌。搭乘地鐵或接駁車前往太空港。請注意看站牌提示，確保搭乘正確路線的班車。

3. 召喚飛船

  1. 抵達太空港後，尋找 Fleet Manager (艦隊管理終端，俗稱叫船終端)。
  2. 在終端上點擊您的飛船旁邊的 Retrieve (召喚) 按鈕。
  3. 注意螢幕上顯示的 Hangar (停機坪) 號碼，例如 Hangar 04。
  4. 前往旁邊的電梯，按 F 選擇前往對應的停機坪。

4. 起飛與離港

進入飛船後，找到駕駛座並按 F 坐下。啟動飛船電源 (按 R 或 U 啟動電源、I 啟動引擎)。

  • 按 F11 打開通訊面板，在 Friends 標籤頁找到 Spaceport Landing Services，點擊通話圖示請求起飛許可。
  • 機庫門打開後，按 Space (空白鍵) 垂直向上平移。
  • 滑鼠控制機頭方向，W/S 控制前進後退，A/D 左右平移，N 收起起落架。

5. 飛往軌道空間站

飛出大氣層後，按 F2 打開星圖 (Starmap)。在您當前星球的軌道上尋找空間站 (例如 New Babbage 上方是 Port Tressler)。點擊空間站並選擇 Set Route (設定路線)。

對準目標點，按住 B 鍵進行量子跳躍 (Quantum Travel)。抵達後，再次使用 F11 呼叫空間站降落服務，根據指示圖標降落並進入空間站，完成您的第一次航行！
