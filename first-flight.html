第一次航行操作手冊
(主城至空間站)

1. 進入個人機庫與基本互動

在最新版本中，登入後您會直接在自己的個人機庫內。您可以按 F4 切換第三人稱視角，按住 Z 自由觀看周遭。

特別注意：長按或按下 F 鍵可以呼叫出互動選單，這是遊戲中最重要的一個互動按鍵，舉凡開門、搭電梯、撿取物品到駕駛飛船，都需要靠它。

2. 使用 ASOP 終端機召喚飛船

  1. 在機庫正前方，您會看到一台發光的 ASOP Terminal (艦隊管理終端機)。
  2. 靠近終端機並對著螢幕按下 F 鍵開始互動。螢幕上會列出您擁有的飛船。
  3. 如果是第一次呼叫該飛船，請點擊飛船名稱右側的 交付 (Delivery) 按鈕，這會將飛船運送到您的機庫。
  4. 交付完成後，再次點擊 召喚 (Retrieve)，機庫中央的升降平台就會將您的飛船升上來。

3. 尋找入口與登機

使用 W A S D 鍵移動，靠近飛船並尋找入口。請注意，不同的飛船有不同的上機結構 (有些是後艙門、有些是側邊梯子或升降梯)。

找到入口後，對著艙門按下 F 鍵開啟。進入飛船內部後，找到駕駛座，對著座椅按下 F 鍵即可坐上駕駛座。

4. 啟動與起飛離港

坐上駕駛座後，請牢記以下三個起飛前最重要的按鍵：

  • U 鍵：啟動飛船電源。
  • I 鍵：開啟飛船引擎。
  • ALT + N：呼叫機庫門開啟 (自動請求起飛許可)。

完成啟動並等待機庫門完全打開後：

  • 按 Space (空白鍵) 垂直向上平移。
  • 滑鼠控制機頭方向，W/S 控制前進後退，A/D 左右平移。
  • 離開機庫後，按 N 鍵收起起落架。

5. 飛往軌道空間站

飛出大氣層後，按 F2 打開星圖 (Starmap)。在您當前星球的軌道上尋找空間站 (例如 New Babbage 上方是 Port Tressler)。點擊空間站並選擇 Set Route (設定路線)。

對準目標點，按住 B 鍵進行量子跳躍 (Quantum Travel)。抵達後，按下 ALT + N 呼叫空間站降落服務，根據指示圖標降落並進入空間站，完成您的第一次航行！
