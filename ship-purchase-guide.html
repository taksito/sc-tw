📦 Hauling (貨運)

專為跑商與物流合約設計。從基礎的小型快遞到星際級巨型貨輪。

  • • 新手：Avenger Titan / Aurora MK2
  • • 進階：Hull A / Cutlass / RAFT
  • • 大型：Hull B / C2 / Hull C

🚀 移動與交通 (Movement)

注重航行速度、遠航能力與多人運輸。適合探險、接駁或快速穿越星系。

  • • 入門推薦：Cutter Rambler
  • • 經典選擇：Aurora MK2 / Avenger Titan
  • • 進階：400i / Constellation

🏗️ Salvage (回收)

從廢棄船隻中提取珍貴材料與組件。當前宇宙中最穩定的賺錢方式之一。

  • • 單人首選：Vulture (禿鷹級)
  • • 團隊作業：Reclaimer (開拓者級)

⛏️ 挖礦 (Mining)

採集地表與小行星帶的稀有礦產。適合喜歡精確操作與工業化流程的玩家。

  • • 單人入門：Prospector (勘探者級)
  • • 團隊多人在線：MOLE
